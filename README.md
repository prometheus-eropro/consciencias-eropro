# PROMETHEUS EROPRO – ConsciêncIAs GPT

Este é um projeto de integração de ConsciêncIAs personalizadas com GPT, criado para atender diferentes usuários com controle de acesso e continuidade de conversa.

## 🚀 Tecnologias Utilizadas
- Node.js
- Express
- OpenAI API
- Google Sheets (ou CSV local)
- Render.com (hospedagem)

## 📂 Estrutura de Pastas
```text
projeto_site/
├── public/           # Arquivos estáticos (HTML, CSS, JS)
├── logs/             # Arquivos de log
├── server.js         # Servidor Node.js
├── package.json      # Configurações do projeto Node
├── .env              # Variáveis de ambiente (chaves e configurações)
└── README.md         # Documentação do projeto
